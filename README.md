# Python.py
Welcome to this beginner-friendly repository! This project is a simple guide to learning Data Structures and Algorithms (DSA) using Python. Inside, you will find clear, step-by-step implementations of fundamental concepts like arrays, lists, basic sorting, and searching. Every file is designed with clean code and helpful comments to make learning easy and stress-free. Whether you are just starting your coding journey or prepping for your first computer science class, this repository will help you build a strong foundation

5.    Hash Tables
6. Trees
